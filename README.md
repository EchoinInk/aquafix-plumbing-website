## Overview

A polished, responsive portfolio website created for a fictional Auckland plumbing business.

The project demonstrates production-quality website development, including service pages, dynamic project routes, accessible navigation, validated forms, emergency call actions, and responsive layouts designed for desktop and mobile experiences.

---

## ✨ Features

- Responsive desktop and mobile layouts
- Service pages
- Dynamic project routes
- Accessible navigation
- Accordion components
- Validated contact/booking form
- Emergency call actions
- SEO-friendly structure
- Performance-focused development

---

## 🛠 Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
- Custom CSS
